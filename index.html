Purpose

Currently the activity log is under development. The aim is to create something to keep track of and manage incoming data received by the Data Linkage Branch of Western Australia.

Background

Currently information about data coming into the branch is stored in a number of spreadsheets across a number of folders. This information is entered in the spreadsheets manually by linkage officers often with instructions which can be ambiguous. It is hoped (mainly by me) that we can create an application to make this easier. This should also make finding information relating to the datasets slightly easier to, and maybe make everything more efficient and transparent.
