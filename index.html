Dlb activity log

A data tracking tool for incoming data into the Data Linkage Branch of Western Australia

Download this project as a .zip file Download this project as a tar.gz file

Purpose

Currently the activity log is under development. The aim is to create something to keep track of and manager incoming data received by the Data Linkage Branch of Western Australia.

Tools

A number of open source tools and technologies have been used to create the activity log. These include:

  • Django
  • BootStrap/Django Bootstrap
  • jQuery
  • pip
  • ... For a full list see requirments
